This allows users to publish their X-Ray Goggles hacks to their own S3
bucket.

This software is still in an embryonic state. Use at your own risk.

## Quick Start

    git clone git://github.com/toolness/custom-hackpub.git
    cd custom-hackpub
    npm install
    npm test
    node app.js
